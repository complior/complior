Key Features

Wrapper-Orchestrator

Wraps any CLI coding agent (Claude Code, aider, OpenCode, bash) as a PTY subprocess. You keep working — Complior adds the compliance layer.

Real-time Compliance Gate

Every file change → background rescan in 200ms → score update → toast notification with exact article reference.

19 Deterministic Checks

AST-based rules — no LLM in compliance decisions. Covers EU AI Act Art. 50, 12, 4, 11, 51-56 and more.

6 auto-fixers for disclosure, marking, logging, docs, metadata, and FRIA — with diff preview before applying.
